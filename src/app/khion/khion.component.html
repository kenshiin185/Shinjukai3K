Kihon grades

Kihon pour la ceinture Jaune

Description:

  1. Oï zuki Jodan / Jodan Age uke + Gyaku zuki
  2. Oï zuki Chudan / Soto uke + Gyaku zuki
  3. Mae geri + Tsuki / Gedan baraï (45°) + Gyaku zuki

Kihon pour la ceinture Orange

Description:

  1. Oï zuki Jodan / Jodan Tate uke (kokutsu dachi) + Gyaku Ura zuki
  2. Oï zuki Chudan / Shuto uke (kokutsu dachi 45°) + Gyaku Shuto uchi
  3. Mae geri + Tsuki / Uchi baraï (kiba dachi 45°) + Gyaku Haïto uchi
  4. Mawashi geri + Uraken / Uchi uke (45°) + Gyaku Teïsho uchi

Kihon pour la ceinture Verte

Description:

  1. Oï zuki Jodan / Jodan Teïsho uke + Gyaku Mawashi Empi
  2. Oï zuki Chudan / Osae uke + Gyaku Nukite
  3. Mae geri + Tsuki / Gyaku Gedan baraï + Fumikomi (kiba dachi)
  4. Mawashi geri + Uraken / Tate Shuto uke (kiba dachi 45°) + Yoko Empi
  5. Yoko geri + Tetsui (kiba dachi) / Otoshi uke (kiba dachi) + Kagi zuki
