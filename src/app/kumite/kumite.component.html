Ippon Kumite Atemi Orange

Description:

  1. Oï zuki Jodan / Jodan Tate uke (kokutsu dachi) + Gyaku Ura zuki
  2. Oï zuki Chudan / Shuto uke (kokutsu dachi 45°) + Gyaku Shuto uchi
  3. Mae geri / Uchi baraï (kiba dachi 45°) + Gyaku Haïto uchi
  4. Mawashi geri / Uchi uke (45°) + Gyaku Teïsho uchi

Ippon Kumite Atemi Verte

Description:

  1. Oï zuki Jodan / Jodan Teïsho uke + Gyaku Mawashi Empi
  2. Oï zuki Chudan / Osae uke + Gyaku Nukite (ou Hira basami)
  3. Mae geri + Tsuki / Gyaku Gedan baraï + Fumikomi (kiba dachi)
  4. Mawashi geri + Uraken / Tate Shuto uke (kiba dachi 45°) + Yoko Empi
  5. Yoko geri + Tetsui (kiba dachi) / Otoshi uke (kiba dachi) + Kagi zuki

  2. Oï zuki Chudan / Osae uke + Kokyu nage (ue)
  3. Mae geri + Tsuki / Gyaku Gedan baraï + Ushiro Ashi baraï, Tsuki
  4. Mawashi geri + Uraken / Tate Shuto uke (kiba dachi 45°) + Haraï goshi, Tsuki
  5. Yoko geri + Tetsui (kiba dachi) / Otoshi uke (kiba dachi) + Irimi nage
